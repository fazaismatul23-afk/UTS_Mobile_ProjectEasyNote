EasyNote Application merupakan aplikasi catatan sederhana yang dibuat untuk memenuhi tugas kelompok UTS. EasyNote adalah aplikasi catatan sederhana yang dirancang untuk membantu pengguna mencatat ide, tugas, dan informasi penting dengan cepat dan praktis. Aplikasi ini memiliki tampilan yang bersih dan mudah digunakan, dilengkapi fitur seperti tambah, edit, hapus, serta auto-save sehingga setiap catatan tersimpan secara otomatis. Dengan EasyNote, pengguna dapat mengelola catatan harian secara efisien tanpa ribet.
